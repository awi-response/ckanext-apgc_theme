{% trans %}The Arctic Permafrost Geospatial Centre (APGC){% endtrans %}

{% trans %}APGC is an Open Access Circum-Arctic Geospatial Data Portal that promotes, describes and visualizes geospatial permafrost data. A data catalogue and a WebGIS application allow to easily discover and view data and metadata. Data can be downloaded directly via link to the publishing data repository.{% endtrans %}

{% trans %}Mission{% endtrans %}

  • {% trans %}APGC provides open access to high-quality scientific data related to permafrost in the Arctic.{% endtrans %}
  • {% trans %}APGC facilitates discovery, view, dissemination, and archival of permafrost geospatial information.{% endtrans %}
  • {% trans %}APGC supports metadata documentation and exchange.{% endtrans %}

{% trans %}Data Criteria{% endtrans %}

{% trans %}New submissions to the catalogue are evaluated for publication using the following criteria.{% endtrans %}

{% trans %}PERMAFROST FOCUS: Data directly or indirectly measures and/or classifies permafrost characteristics in the Arctic, Antarctica, or mountain permafrost areas, for example soil temperatures, ground ice content, land cover, topography and many more.{% endtrans %}

{% trans %}HIGH SIGNIFICANCE AND IMPACT: The data addresses an important problem or critical barrier to progress in the field of permafrost research. It provides, for example, circum-arctic coverage, is important to many different disciplines, and shows new or improved measurements.{% endtrans %}

{% trans %}OPEN ACCESS: Metadata and data are accessible and published in a repository.{% endtrans %}

{% trans %}HIGH QUALITY: Data is validated. Metadata reports an accuracy assessment.{% endtrans %}

{% trans %}METADATA: Metadata adequately describes the processing of the data and the data is reproducible with the given metadata.{% endtrans %}


{% trans %}Funding and Support{% endtrans %}

{% trans %}APGC is maintained by the Permafrost Research section of the Alfred Wegener Institute (AWI), Helmholtz Centre for Polar and Marine Research. The WebGIS application has been developed by the AWI Computing and Data Center. The APGC has been initiated by the ERC project Rapid Permafrost Thaw in a Warming Arctic and Impacts on the Soil Organic Carbon Pool (PETA-CARB), and the European Space Agency (ESA) project GlobPermafrost.{% endtrans %}
